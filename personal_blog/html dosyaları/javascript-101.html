Yeni Başlayanlar İçin JavaScript 101

JavaScript, web sayfalarına dinamik özellikler eklemek ve etkileşimli web uygulamaları oluşturmak için kullanılan güçlü bir programlama dilidir. HTML ve CSS'in ötesinde, JavaScript, web sayfalarındaki öğeleri yönetmek, kullanıcı etkileşimine tepki vermek ve daha fazlasını yapmak için kullanılır. Web tarayıcıları tarafından çalıştırılan JavaScript, her türlü web projesinin temel bir parçası haline gelmiştir.

JavaScript öğrenmeye başlamak için önce dilin temel yapı taşlarını anlamanız gerekir. Bu yazıda, JavaScript'in temellerine odaklanacağız ve basit bir projeyi nasıl geliştireceğinizi adım adım göstereceğiz. JavaScript dilinde temel öğeler arasında değişkenler, fonksiyonlar, döngüler ve koşullar yer alır. Bunları anlamak, daha karmaşık kod yapıları oluşturmanızı sağlayacaktır.

1. Değişkenler ve Veri Türleri

Değişkenler, verileri saklamak ve işlemek için kullanılan yapılardır. JavaScript'te değişkenler, let, const ve var anahtar kelimeleriyle tanımlanabilir. En yaygın kullanılan değişken tanımlama şekli lettir, çünkü bu, değişkenlerin sadece bulunduğu blok içinde geçerli olmasını sağlar.

let age = 25;  // Bir sayı değeri
let name = "Bilal";  // Bir metin değeri
let isJavaScriptFun = true;  // Bir boolean (doğru/yanlış) değeri

2. Fonksiyonlar

Fonksiyonlar, belirli bir işlemi yerine getiren kod bloklarıdır. Fonksiyonlar, tekrar kullanılabilir kod yazmanıza olanak tanır. JavaScript'te fonksiyonlar şu şekilde tanımlanır:

function greet() {
    console.log("Merhaba, JavaScript!");
}
greet();  // Fonksiyon çağrısı

Yukarıdaki örnekte, greet adlı bir fonksiyon tanımlandı ve daha sonra çağrılarak konsola mesaj yazdırıldı. Fonksiyonlar, parametre alabilir ve değer döndürebilir.

3. Döngüler

Döngüler, aynı işlemi birden fazla kez yapmanıza olanak tanır. JavaScript'te en yaygın döngüler for, while ve do...while döngüleridir. İşte for döngüsünün bir örneği:

for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4 değerlerini yazdırır
}

4. Koşullar

Koşul ifadeleri, belirli bir durumun gerçekleşip gerçekleşmediğine göre farklı kod parçalarının çalıştırılmasını sağlar. JavaScript'te en yaygın koşul ifadesi if...else yapısıdır. İşte bir örnek:

let number = 10;
if (number > 5) {
    console.log("Sayı 5'ten büyüktür.");
} else {
    console.log("Sayı 5'e eşit veya küçüktür.");
}

Yukarıdaki örnekte, number değişkeninin değeri 5'ten büyük olduğu için ilk koşul çalışır ve konsola "Sayı 5'ten büyüktür." mesajı yazdırılır.

5. Etkileşimli Uygulamalar

JavaScript'in gücü, etkileşimli uygulamalar yaratabilmesindedir. Kullanıcıların etkileşimde bulunduğu web sayfalarında, JavaScript sayesinde öğeleri değiştirebilir, animasyonlar ekleyebilir ve kullanıcı girdilerine tepki verebilirsiniz. Örneğin, bir formu gönderdiğinizde veya bir butona tıkladığınızda bir işlem başlatabilirsiniz.

Örnek Kod: "Merhaba" Yazdırma

let greeting = "Merhaba, JavaScript!";
console.log(greeting); // Konsola "Merhaba, JavaScript!" yazdırır

Yukarıdaki basit JavaScript kodu, konsola "Merhaba, JavaScript!" mesajını yazdırır. Bu kodu kendi bilgisayarınızda deneyerek ilk adımı atabilirsiniz! Herhangi bir hata alırsanız, kodunuzu dikkatlice gözden geçirdiğinizden emin olun. Kodun doğru şekilde çalıştığından emin olduktan sonra, daha karmaşık projelere geçebilirsiniz.

6. JavaScript İle Basit Proje

JavaScript’i öğrenmek için projeler yapmak harika bir yoldur. İlk başta basit bir hesap makinesi uygulaması yapabilirsiniz. Bu, kullanıcı girişini almayı, matematiksel işlemler yapmayı ve sonuçları göstermeyi öğrenmenizi sağlar.

Örneğin, kullanıcıdan iki sayı alıp bu sayıları toplamak için şu basit kodu yazabilirsiniz:

let number1 = prompt("Birinci sayıyı girin:");
let number2 = prompt("İkinci sayıyı girin:");
let sum = parseFloat(number1) + parseFloat(number2);
alert("Toplam: " + sum);

Bu basit uygulama, kullanıcının girdiği iki sayıyı toplar ve sonucu ekrana yazdırır. Kullanıcıdan veri almak için prompt() fonksiyonunu, sonucu göstermek için ise alert() fonksiyonunu kullanıyoruz.
